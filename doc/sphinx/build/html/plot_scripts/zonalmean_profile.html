zonalmean_profile¶

zonalmean_profile(wks_in[1], source, varname[1]: string)¶
Parameters:
  • wks_in (integer) – workstations (graphic object or default will be used).
  • source (integer) – data to be plotted or a NetCDF filename with data.
  • varname (string) – variable name in the file.
Source prototype
source[,] source!0 = plev source!1 = lat
Return value
A graphic variable
Description
Draws a pressure-latitude plot
