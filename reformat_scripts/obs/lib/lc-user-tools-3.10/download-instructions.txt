The ESACCI-LANDCOVER reformatting script requires the CCI-LC User Tools.

Download instructions:

    1) Goto http://maps.elie.ucl.ac.be/CCI/viewer/index.php and click on "Download data".
    2) Select "Climate Research Data Package (CRDP)".
    3) Register with your name and email.
    4) Search for "User tool" and download the tools.
    5) unpack the source code files into the directory specified in the reformatting script
       reformat_scripts/obs/reformat_obs_ESACCI-LANDCOVER.py (variable path2lctool)
